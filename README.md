# CS409PC - Node.js Lab Experiments

This repository contains all Node.js lab experiments developed as part of the CS409PC course.

## 📁 Folder Structure

* **EXP01/** – Introduction to Node.js and basic programs
* **EXP02/** – File system operations
* **EXP04/** – Web server and routing concepts

*(More experiments will be added as the course progresses.)*

---

## 🚀 Technologies Used

* Node.js
* JavaScript
* Express.js (if used in later experiments)

---

## ▶️ How to Run

1. Install Node.js from https://nodejs.org
2. Clone the repository:

   ```bash
   git clone https://github.com/SaheelHussain/CS409PC-Node-Js-24AG1A05BY.git
   ```
3. Navigate to any experiment folder:

   ```bash
   cd EXP01
   ```
4. Run the file:

   ```bash
   node filename.js
   ```
